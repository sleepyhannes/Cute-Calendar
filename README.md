# 🌸 Cute-Calendar

A cute desktop calendar application built with HTML, CSS, JavaScript, and ElectronJS. This project features a custom **Spider Lily** theme.

## Project Overview

This project serves as a lab activity designed to combine aesthetic front-end development with functional utility. The application provides a date display using custom assets and custom typography.

## Design & Aesthetic

The project focuses on a aesthetic style for digital interfaces.
- **Theme:** Custom Spider Lily assets
- **Typography:** Libre Bodoni Regular
- **Special Features:** Draggable window interface using `-webkit-app-region`

## What's in this repo

| File / Folder | Purpose |
| --- | --- |
| `index.html` | The core structure of the calendar widget |
| `style.css` | All aesthetic configurations and layout alignments |
| `script.js` | Frontend JavaScript controlling calendar logic |
| `main.js` | The Electron main process script — creates the desktop window |
| `package.json` | Project metadata and dependencies |
| `.gitignore` | Files and folders ignored by Git |
| `assets/` | Contains all images, backgrounds, and font files (TTF) |

## How to use

### 1. Clone This Repo

```
git clone https://github.com/hannesdssddss/Cute-Calendar.git
cd my-cute-calendar
```

### 2. Install Node.js

Make sure you have Node.js installed, then run:

```
npm install
```
This installs Electron and any other modules required to run the app.

### 3. Run the App Locally

```
npm run start
```

This will open your Cute-Calendar as a desktop application powered by Electron.
